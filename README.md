# Ebba Portfolio Website

Single-page personal portfolio website based on Ebba Adolphson's CV.

## Contents

- `index.html` - page structure and content
- `styles.css` - layout, typography, color system, and responsive design
- `script.js` - reveal-on-scroll animation and footer year

## Run locally

Open `index.html` directly in your browser.

## Add profile photo

Place your image at `assets/ebba-profile.jpg`.
The hero section will automatically render it.
If no image exists, the site shows a styled `EA` fallback tile.

## Deploy options

- GitHub Pages
- Netlify
- Vercel
